## ğŸ¨ Customization

### Change Colors
Edit the color scheme in `style.css`:
```css
/* Main gradient colors */
--primary-cyan: #00ffff;
--primary-green: #00ff88;
--dark-bg: #0a0e27;
```

### Update Content
- **Profile Info**: Edit text in `index.html` sections
- **Projects**: Update project cards and create new detail pages
- **Images**: Replace images in the `image/` folder
- **Skills**: Modify skill cards in the Skills section

### Add New Projects
1. Create a new HTML file (e.g., `new-project.html`)
2. Copy structure from existing project pages
3. Update content, images, and links
4. Add project card to `index.html`
